Mini-Hobby
====
### 팀명 : 열심히하조 😎
> 2021/02/03~2021-02-15 Toy Team Project (5조)

##### 분야별 1:1 강습을 원하는 학생과 서비스를 제공하는 강사를 연결해주는 서비스를 제공하는 프로그램입니다.
<br/>

> 팀원 구성 
 - 김현정 (조장)
 - 박세라 
 - 이강산
 - 이영준

> Skill
 - Java
 - Swing
 - Oracle
 - Git/Github
<br/>

> 담당기능
- 회원가입-로그인-회원정보수정-탈퇴, 학생 - 마이페이지, 학생-메인
